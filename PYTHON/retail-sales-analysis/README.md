# 📊 Retail Sales Analysis

A real-world Data Analysis project focused on understanding sales performance, profitability, customer trends, and discount impact using the Superstore dataset.

---

## 🚀 Project Overview

This project analyzes **10,000+ retail transactions** to uncover valuable business insights that can help improve decision-making and revenue growth.

The analysis was performed using **Python** and industry-standard data analysis libraries.

---

## 🛠️ Tools & Technologies Used

- 🐍 Python  
- 🐼 Pandas  
- 🔢 NumPy  
- 📈 Matplotlib  
- 🎨 Seaborn  
- 📓 Jupyter Notebook  

---

## 📂 Dataset Information

The dataset contains:

- 🧾 Order Details  
- 👤 Customer Information  
- 🌍 Region / City / State Data  
- 📦 Product Categories  
- 💰 Sales & Profit  
- 🏷️ Discounts  
- 📅 Order Dates
